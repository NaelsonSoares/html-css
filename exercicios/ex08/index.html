Nesta frase, temos um termo em itálico usando a tag I (não semântica).

Texto marcado

Texto deletado

Podemos marcar um texto como excluído usando a tag DEL para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto insrido com a tag INS para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o sublinado com a tag U (não semântica).

Texto sobrescrito

Para inserir coisas do tipo x20+3, usando a tag SUP.

Texto subscritos

Para inserir coisas do tipo H2O, usando a tag SUB.
